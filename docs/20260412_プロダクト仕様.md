# Multi Turtle CLI Deck プロダクト仕様

## 1. 目的
- 複数の CLI エージェントを一画面で整然と運用する。
- IDE ほど情報過多にせず、CLI 単体よりも停止検知、切り替え、再利用をしやすくする。
- コードを直接見続けなくても、複数タスクの進行状況と危険信号を判断できるようにする。

## 2. プロダクト定義
- Terminator の分割運用に近い体験を、Web ベースのタスクデッキとして再構成する。
- 各ペインは CLI セッション、対象ワークスペース、共有コンテキスト、進行状態を持つ。
- 生 PTY を前提にせず、CLI の品質を維持したまま、操作性だけを UI が補完する。

## 3. 実装済みコア機能

### 3.1 ペイン単位の実行
- Codex CLI / GitHub Copilot CLI / Gemini CLI の切り替え
- モデル選択
- 推論レベル選択
- 実行権限レベル選択
  - Codex は balanced で --full-auto、max で --dangerously-bypass-approvals-and-sandbox を使う
  - Gemini は auto_edit / yolo 相当の切り替え
  - Copilot は allow-all-tools / allow-all 相当の切り替え
- Codex Fast モード切り替え

### 3.2 ワークスペース操作
- ローカルワークスペース選択
- ローカル選択の初期位置は、前回ブラウズ位置、ペインの現在ワークスペース、OS別デフォルトの順で決める
  - Windows は存在するドライブを優先する
  - Linux / macOS は Home を優先する
  - このツール自身のワークスペースは初期位置として自動優先しない
- ネイティブフォルダダイアログからのローカル追加
- VS Code で開く
- SSH ホスト入力
- SSH 先のワークスペース候補検出
- SSH 先にインストール済みの CLI 検出
- Remote-SSH URI で VS Code を開く

### 3.3 会話継続
- 各ペインに CLI セッション ID を保持
- 同一ペインで次回実行時に resume を優先
- ページ再読み込み後も状態を localStorage に保持
- 実行中に切断されたセッションは前回の実行は切断されましたとして可視化

### 3.4 情報引き継ぎ
- ペインの結果を共有コンテキストとして保存
- 他ペインから必要な共有コンテキストを選択して次回実行に添付
- 自動共有トグルで、完了結果を即座に共有ストックへ反映

### 3.5 実行可視化
- ペイン枠と LED の色で状態表示
  - 待機: ニュートラル
  - 実行中: シアン
  - 完了: グリーン
  - 確認待ち: イエロー
  - エラー: レッド
- Live Output にストリーミング応答を表示
- Activity にツール実行、状態変化、stderr を表示
- Conversation にユーザー / アシスタント / システム履歴を保持
- 一定時間更新がない実行を停滞として検知
- ヘッダー直下に warning / error notice を出して確認が必要な実行を見落としにくくする

### 3.6 補助操作
- ペイン複製
- 会話リセット
- 最新応答コピー
- 実行停止
- レイアウト切り替え
  - 2x2
  - 3列
  - Focus

## 4. バックエンド設計
- POST /api/run/stream
  - NDJSON でイベントを返す
  - assistant-delta / status / tool / stderr / session / final / error
- POST /api/run/stop
  - ペイン単位で実行停止
- POST /api/ssh/workspaces
  - SSH 先ワークスペース探索
- POST /api/ssh/inspect
  - SSH 先 CLI 検出
- POST /api/system/open-vscode
  - 対象ワークスペースを VS Code で開く

## 5. 現在の UI 方針
- 生ターミナルではなく、Composer + Live Output + Activity の三層構成
- 直接ターミナルを触るより、タスクを流し込んで監視する方に最適化
- 停止検知、共有、切り替え、再開を UI が担当し、CLI の回答品質は CLI 本体に委ねる

## 6. 次の拡張候補

- 共有コンテキストのタグ付けと検索
- ペイングループごとのテンプレート保存
- 実行キューと依存関係
- 差分プレビューや成果物比較ビュー
